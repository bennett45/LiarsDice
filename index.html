Liar's Dice

Teaser

Liar's Dice is a classic dice game that deals with guessing and calling your opponents bluffs. This is a playable AI for Liar's Dice.

What

This is a Playable game of Liar's Dice. It has an AI that uses a monte carlo tree search algorithm to solve what move it should use. It also has an AI that uses some randomness to bluff.

Why

I have played this game for years and it is a family favorite of mine. I wanted to see if I could create an AI that would be able to be competitive with humans.
